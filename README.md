# chat-esl
本项目是一个结合人工智能与语言学习的多功能平台，旨在帮助用户通过自然对话高效学习英语。系统内置 AI 聊天模块，可实时纠正语法错误、解释语言用法，并提供多种学习模式，如自由对话、纠错讲解、词汇扩展和模拟考试。用户可在轻松的交流环境中逐步提升听说读写能力。同时，项目集成了强大的工具库 Utils，支持 AES-GCM 与 RSA 加密、文件与时间管理、字符串处理等功能，确保学习数据的安全与本地化存储，是一款集智能学习与数据安全于一体的现代化英语学习助手。
# 🧠 AI English Tutor & Utils Library

> 一个支持 **AI 实时英语学习** 与 **加密安全工具集** 的综合项目。  
> 由 **Simon** 定制，基于 **Python 3.13** 构建。

## 🚀 快速开始

### 🧩 环境要求
- Python **3.10+**（推荐 3.13）
- 系统已安装 pip

### 📦 安装依赖
```bash
pip install -r requirements.txt
```

或手动安装核心依赖：
```bash
pip install cryptography pycryptodome
```

### ▶️ 运行项目
```bash
python app.py
```

运行后打开 Web 界面（或命令行终端）即可与 AI 进行实时英语聊天练习。

## 💬 AI 聊天式英语学习

本项目的核心功能是通过 **AI 实时互动对话** 学习英语。  
你可以像和真人老师聊天一样练习口语、写作和语法。  
AI 会自动分析语法结构、提供反馈和替代表达。

---

### 🌍 1. 实时纠错与语法讲解

AI 在聊天时能自动检测语法错误，并即时反馈。  

> **示例：**  
> 👤: I am interesting in play football.  
> 🤖: You should say “I am interested in playing football.”  
> ✏️ Explanation: “interested” 表示“感兴趣的”，后面要接动名词（V-ing）。

**优点：**
- 自动区分语法错误与自然表达；
- 提供中英文解释；
- 支持逐句分析与翻译。

---

### 🧠 2. 多模式学习体验

支持多种学习模式：

| 模式 | 功能说明 |
|------|-----------|
| 🗣 **Free Chat** | 自由英文对话，不限话题 |
| 🧩 **Correct & Explain** | 自动纠错 + 语法分析 |
| 📚 **Vocabulary Mode** | 词汇拓展与例句练习 |
| 📝 **Exam Mode** | 模拟口语与写作考试 |

你可以通过命令切换模式，让学习更有针对性。

---

### 🪄 3. 智能语境学习

AI 会根据你的表达水平自动调整对话难度。  
它能：
- 根据上下文推荐更自然的表达；
- 动态切换语法层级（B1 → C1）；
- 模拟不同场景（面试、演讲、日常会话等）。

> 这就像一个**会记忆你语言习惯的私人英语教练**。

---

### 📈 4. 学习记录与复盘

系统自动记录所有对话历史，可随时回顾学习过程。  
支持：
- 自动导出聊天记录（JSON / Markdown）；
- 分析语法错误出现频率；
- 生成个性化词汇清单；
- 自动统计学习时长与改进趋势。

## 🔐 内置安全工具库：`Utils`

项目集成了一个强大的工具类 **`Utils`**，  
提供常用系统函数、文件管理与加密功能。

### 功能总览

| 分类 | 功能说明 |
|------|-----------|
| ⏰ 时间函数 | 时间戳、ISO 时间、字符串 ↔ 时间戳 转换 |
| 📂 文件操作 | 自动创建目录、读写文件、JSON 读写 |
| 🔤 字符串工具 | Slugify、分段、URL 编解码、字符串截断 |
| 💾 系统信息 | 环境变量读取、主机名、计时器等 |
| 🔒 AES-GCM | 固定 IV 加密（兼容 Java），支持 AES-128 / 192 / 256 |
| 🔑 RSA | 密钥对生成、公钥加密 / 私钥解密（Base64 DER） |

---

### ⚙️ AES-GCM 加密说明

使用 `cryptography.hazmat.primitives.ciphers.aead.AESGCM` 实现，  
支持 **AES-128 / AES-192 / AES-256**。  
- 固定 IV：`b"1234567890ABdefg"`（与 Java AES-GCM 对齐）  
- 输出为 Base64 编码密文（包含 tag）  
- 支持附加认证数据（AAD）验证完整性  

---

### 🔑 RSA 加密说明

基于 `pycryptodome` 实现，默认密钥长度 1024 bit。  
可生成：
- 公钥（X.509 格式，Base64 DER）  
- 私钥（PKCS#8 格式，Base64 DER）  

支持：
- 公钥加密  
- 私钥解密  
- 自定义 key 长度（2048 / 4096）
